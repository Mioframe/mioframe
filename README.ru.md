# Self Base

Это приложение для тех, кто хочет управлять своими данными независимо, имея полный контроль над их хранением.

## Реализовано и планы:

- [x] Использование Conflict-Free Replicated Data Types (CRDT)
- [x] Хранение в браузере
- [x] Локальное хранение
  - [x] на ПК
  - [x] на мобильных устройствах
- [ ] Облачное хранение
  - [ ] Синхронизация с Google Drive
- [ ] Данные в формате JSON
  - [ ] Редактирование с помощью [vanilla-jsoneditor](https://github.com/josdejong/svelte-jsoneditor)
- [x] Базы данных
  - Свойства данных
    - Добавление свойств:
      - [x] Строки (string)
      - [x] Числа (number)
      - [x] Логические значения (boolean)
      - [ ] Дата и время
      - [ ] Списки выбора
      - [ ] Ссылки
      - [ ] Связи с другими таблицами
      - [ ] Вычисляемые свойства
    - [ ] Удаление свойств
    - [ ] Редактирование свойств
  - Записи данных
    - [x] Добавление записи
    - [ ] Удаление записи
    - [x] Редактирование записи
  - Представление данных
    - [x] Добавление представления
    - [x] Переименование представления
    - [x] Удаление представления
    - [x] Сортировка представлений
    - [ ] Сортировка данных по значениям
    - [ ] Ручная сортировка
    - [ ] Фильтрация по значению
    - [x] Табличный вид
      - [ ] Скрытие столбцов
      - [ ] Сортировка столбцов
    - [ ] Галерея карточек
- [ ] RichText документ

Мы стремимся создать универсальный инструмент для управления данными, предоставляя вам свободу выбора и полный контроль.
