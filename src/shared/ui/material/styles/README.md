# Material styles

This directory mirrors the official Material `Styles` navigation.

Canonical domains include:

- `color`;
- `elevation`;
- `icons`;
- `motion`;
- `shape`;
- `typography`.

Create a runtime domain only when current component work proves a shared contract. Until migration, existing owners remain documented as legacy dependencies.

Each implemented domain records official evidence, implemented and omitted capability, known issues, affected consumers, verification, and review status in a local `README.md`. An independent review may add a colocated `AUDIT.md`.

Do not pre-create empty implementation directories or wrap generic utilities merely to satisfy this map.
