# Domain: facts

## Purpose
Contains stable project-level facts, technical choices, and operating preferences that should be easy to recall later.

## Scope
Included in this domain:
- Technology choices and architecture facts
- Testing expectations and environment constraints
- Project-wide configuration and operational preferences

Excluded from this domain:
- Personal user profile facts
- Detailed feature implementation notes

## Usage
Use this domain for concise project facts and durable working conventions.
