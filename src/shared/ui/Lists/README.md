# Lists

Material 3 / Material 3 Expressive List component family for `src/shared/ui/Lists`.

Mioframe Lists target the current Material 3 Expressive contract. The legacy `baseline` list style is intentionally unsupported and has been removed from the runtime API.

This existing family contract remains authoritative for the current legacy implementation until the Lists family is assessed and migrated under `docs/material-3/component-adapter.md`.

Do not treat this retained contract as the future m3e adapter contract. During migration, preserve its confirmed consumer scenarios, classify renderer viability, create the canonical family README under `src/shared/ui/material/components/<family>`, and remove this legacy owner only after complete consumer migration.
