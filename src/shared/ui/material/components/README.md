# Material components

This directory mirrors the official Material `Components` navigation.

Each implemented family uses the official documentation slug:

```text
m3.material.io/components/<slug>
→ src/shared/ui/material/components/<slug>
```

## Families

- [Buttons](./buttons/README.md) — active Button-family migration and alignment.

Do not create empty family directories. Add a family when implementation or migration begins.

Each family owns its current implementation documentation in `README.md` and its latest independent review in `AUDIT.md`.
